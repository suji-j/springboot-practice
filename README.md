## 🍀 Springboot 입문
이 리포지토리는 Springboot를 공부한 내용을 정리한 공간입니다.  
스프링 부트, 웹 MVC, 그리고 DB 접근 기술을 코드 예제를 통해 배우는 과정으로 구성되어 있습니다.

<br/>

### 🔍 순서
(1) 스프링 프로젝트 생성 <br/>
(2) 스프링 부트로 웹 서버 실행 <br/>
(3) 회원 도메인 개발 <br/>
(4) 웹 MVC 개발 <br/>
(5) DB 연동 - JDBC, JPA, 스프링 데이터 JPA <br/>
(6) 테스트 케이스 작성
